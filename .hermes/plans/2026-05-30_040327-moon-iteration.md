## Initial Orbit Convention

Current Earth starts at:

```c
position_m = {0.0, 0.0, SOLAR_EARTH_PERIHELION_M};
velocity_mps = {-SOLAR_EARTH_PERIHELION_SPEED_MPS, 0.0, 0.0};
```

Add the Moon at geocentric perigee with the same prograde angular-momentum orientation as the planets (`r x v` has negative Y):

```c
Moon Earth-relative position = {SOLAR_MOON_PERIGEE_M, 0.0, 0.0};
Moon Earth-relative velocity = {0.0, 0.0, SOLAR_MOON_PERIGEE_SPEED_MPS};

Moon absolute position = Earth position + Moon Earth-relative position;
Moon absolute velocity = Earth velocity + Moon Earth-relative velocity;
```

This keeps the Moon close to Earth in the XZ plane and lets the existing N-body integrator apply Sun/Earth/Moon gravitational effects without adding hierarchy code.

Important testing note: do **not** assert that the Moon's absolute acceleration points primarily toward Earth. In heliocentric coordinates near Earth, solar acceleration on the Moon can be larger than Earth's direct acceleration. Instead test Earth-relative state and bounded Earth-Moon separation over short/coarse windows.

## Risks and Tradeoffs

- **Moon stability with current app timestep:** `time_scale = SOLAR_DAY_SECONDS` and variable 60 FPS means one simulation step is about 1440 seconds. That may be acceptable for the Moon, but if orbit visibly jitters or tests reveal drift, plan a follow-up substep task instead of lowering physical correctness expectations silently.
- **Fixed Sun simplification:** The Moon/Earth system will pull on each other, but the Sun remains fixed. That is acceptable for this milestone and should be documented.
- **No barycenter yet:** Real Earth-Moon initialization would place both around their barycenter. This plan intentionally uses Earth state + Moon offset for minimum iteration scope.
- **Visual size/distance:** The Moon's real render radius is tiny and Earth-Moon render distance is only about `0.026` units at current scale. Illustrative radius clamping is required to see it; real-scale mode may make it effectively invisible, which is expected.
- **Absolute acceleration tests:** Solar acceleration dominates the Moon's absolute acceleration, so tests must focus on relative Earth-Moon invariants.
